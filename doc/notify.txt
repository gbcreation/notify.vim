*notify.txt* Plugin to send notification to the user trough the OS notification system

Author:  Gianluca Arbezzano <gianarb@gmail.com>
Author:  Lorenzo Fontana <fontanalorenzo@me.com>


SEND NOTIFICATIONS                                              *notify-notifications*

Calling the emitNotification function
>
    :call notify#emitNotification("Title", "Body")
<

Using the Notify command
>
    :Notify title body
<
